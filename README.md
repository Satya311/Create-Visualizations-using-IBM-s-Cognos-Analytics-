# Create-Visualizations-using-IBM-s-Cognos-Analytics-

Create Visualizations using IBM's Cognos Analytics 
The objective of this part of the Final Assignment is to analyze the historical trends in car sales for SwiftAuto Traders. The goal is to provide insights on car sales and profits for each dealer.
# Installation — Step-by-step instructions on setting up the environment

To access the dataset for the final assignment, navigate to the Cognos Analytics platform. Begin by selecting “Team content” from the navigation panel. Next, choose “Samples.” Within the Samples section, locate the dataset under “By industry > Automotive > Data.” The specific data module you need is named “Auto group data module.” Right-click on this module and opt for the “Create Dashboard” option to load.
